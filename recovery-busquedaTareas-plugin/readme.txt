Para modificar el máximo de tareas a prorrogar maxivamente añadir al devon.properties: (por defecto 500 tareas)
pluginBusquedaTareasMaximoTareasProrrogar=500

Para modificar el plazo máximo para prorrogar tareas incluir en el devon.properties: (por defecto 90 días)
pluginBusquedaTareasDiasPlazoMaximo=90
